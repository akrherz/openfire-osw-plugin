Roadmap

Last updated - 12 April 2010

This roadmap illustrates what core features we're working on. Priorities may change as we go along, but we will keep this overview up to date.

Release Features
v0.6 - 12 April 2010 Real-time status updates
  • Fetch latest updates plus new in real-time
  • With avatar, chat availability status and timestamp
  • Multiple picture attachments per status update
  • Optional list based privacy on a per item basis
  • Separate 'To' field
  • Retrieve profile of author or 'To' recipient

Cross domain contacts
  • Overview of contacts with following status and lists
  • (Un)follow contacts across domains
  • Search new contact based on jid
  • Add to and remove contacts from lists
  • Retrieve profile of contact

Cross domain profile
  • Get profile summary
  • Get 20 last status updates of individual
  • Get full profile
  • Edit your own profile
Unplanned General
  • On site/device registration

Real-time status updates
  • Get all past status updates
  • Add location and tags to status update
  • Share status updates
  • Comment on status updates
  • Liking status updates
  • Delete status updates
  • Add video embed, audio and link attachments

Cross domain contacts
  • Add relations
  • Filter contacts

Cross domain profile
  • Get all previous status updates of individual
